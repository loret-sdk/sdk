# Contributing

## Getting Started

1. Fork the repo and clone it
2. Install dependencies: `pnpm install`
3. Run tests: `pnpm --filter @loret/sdk test`
4. Run type checks: `pnpm --filter @loret/sdk typecheck`

## Pull Requests

- One feature or fix per PR
- Include tests for new behavior
- Run `pnpm --filter @loret/sdk validate` before submitting

## Issues

Use GitHub Issues for bug reports and feature requests.
